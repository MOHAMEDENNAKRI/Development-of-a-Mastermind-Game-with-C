# 🎮 C Number Guessing Game

A console-based number guessing game written in C, featuring difficulty levels, animations, a score system, and a basic leaderboard.

## 📜 Description

This game challenges players to guess a randomly generated number within a specified range. Players can select from three difficulty levels:
- Easy (1-20)
- Medium (1-50)
- Hard (1-100)

It includes:
- Colorful terminal interface
- ASCII animations
- Score tracking and saving
- Leaderboard display
- Joker (Hint) option

## 🚀 Features

- 🧠 Multiple difficulty levels
- 🎨 Animated menus and colorful interface
- 🏆 Score tracking saved to a file
- 📝 Leaderboard display
- 🧩 Hint system to help players guess better
